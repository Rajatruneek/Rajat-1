# Rajat-1
A simple static CRM website prototype for customer data entry with WhatsApp sharing, Excel export, and Word export features.

## Files
- `index.html` — main website page
- `styles.css` — website styling
- `script.js` — form handling, storage, export, and WhatsApp integration

## How to use
1. Open `index.html` in your browser.
2. Enter customer details in the form.
3. Use the buttons to save the entry, send via WhatsApp, export to Excel (`.csv`), or export to Word (`.doc`).
4. Saved entries are shown in the table below the form.

## Notes
- This is a frontend-only demo. Data is stored in browser local storage.
- WhatsApp send opens a link to WhatsApp Web or the mobile app.
- Excel export downloads CSV data that can be opened in Google Sheets or Excel.
- Word export downloads a `.doc` file with saved customer details.
